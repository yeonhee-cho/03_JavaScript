조건문(condition)

프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류
작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if(만약에...)


    if(조건식) {
        조건식이 true인 경우 실행할 코드;
    }

* 조건식: 결과가 true나 false인 식(연산)

* 조건문: 조건식의 결과에 따라 실행을 제어하는 구문(if, switch문)

입력받은 수가 양수인지 음수인지 검사

if(만약에...) - else(아니면) 문


    if(조건식){
        조건식이 true일 경우 수행할 코드;
    } else {
        if 조건식이 false인 경우 수행할 코드;
    }

* else 구문에는 조건식을 작성하는 ()가 없다!!

입력받은 수가 양수인지 음수인지 검사

if(만약에) - else if(아니라면 이렇게, 저렇게) - else(모두 아니면)


    if(조건식1){
        조건식1이 true인 경우에만 실행;
    } else if(조건식2){
        앞선 if의 조건이 false이면서
        else if의 조건식이 true인 경우;
    } else {
        모든 if, else if가 false인 경우 실행;
    }

*else if 구문은 원하는 만큼 작성 가능
* 앞선 if / else  if 문 조건이 false인 경우 
  이후 else if 에서 해당 범위는 제외

어린이, 청소년, 성인 구분하기

나이를 입력받아
0 ~ 13세 : 어린이
14 ~ 19세 : 청소년
20 ~ 150세 : 성인
0미만 150초과 : 잘 못 입력하셨습니다.
미입력 : 값을 입력해주세요.
