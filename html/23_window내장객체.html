window 내장 객체

팝업창에서 읽어갈 값 :
팝업창에서 작성한 값 : 
- 브라우저 창 자체를 나타내는 객체

- 자바스크립트 모든 객체의 최상위 부모

- window의 자식 객체
  1) DOM(Document Object Model) : document
     (브라우저에서 내용(페이지) 출력 부분)
  2) BOM(Browser Object Model) : location, history, screen, navigator
     (DOM을 제외한 브라우저 나머지 부분 -> 기능 부분)

[window].alert("문자열") : 확인 버튼만 존재
[window].prompt("문자열") : alert창에서 텍스트를 입력할 수 있는 존재
[window].confirm("문자열") : 확인 취소 버튼이 존재

시간 지연 함수

- window에서 제공하는 시간 관련 함수들은 모두 비동기로 동작

  동기 : 앞 코드가 수행이 종료되어야 다음 코드 수행
  비동기 : 코드 해석(작업)이 작성 순서대로 수행되지 않고, 여러 작업이 동시에 수행되는 것
           -> 먼저 읽은 코드가 수행이 완료되는 것을 기다리지 않고 바로 다음 코드를 수행하는 것

[window].setTimeout(함수, 지연시간(ms))

- 지연 시간이 지난 후 함수를 1회만 수행하도록 하는 widow 객체 내장 함수
  타이머(시간이 지정된)

[window].setInterval(함수, 지연시간(ms))

- 지연 시간이 지날 때 마다 함수를 계속 수행하는 widow 객체 내장 함수
  타이머(시간이 지정된)

[window].clearInterval( setInterval() )

- setInterval()로 인해 생성된 인터벌을 삭제하는 함수

window.open("URL", "팝업창 이름", "팝업창 특성(옵션)")

url : 새 창에서 요청할 URL
팝업창 이름 : 
              사용자 임의지정 : 새 창 이름 지정
              _blank : 새 탭 또는 새 창(기본값)
              _self : 현재 탭 또는 현재 창
              _parent : 부모페이지
              _top : 최상위 페이지
팝업창 특성 : 
              popup : 최소한의 팝업창 사용
              width or innerWidth : 스크롤 막대를 포함하여 콘텐츠 영역의 가로 너비 지정
                                    최고 필수값 100
              height or innerHeight : 스크롤 막대를 포함하여 콘텐츠 영역의 세로 너비 지정
                                      최고 필수값 100
