웹 스토리지 - 쿠키, 세션, 로컬

쿠키(cookie)

서버와 주고 받는 작은 데이터
로그인 정보 저장에 많이 사용
-> 직접 적으로 로그인 정보를 저장하는 것이 아니라
로그인을 할 수 있는 토큰(=인증키)를 저장 후 로그인 정보를 가져 올 수 있도록 설정

세션(session)

탭을 닫으면 사라지는 데이터, 임시 저장용으로 많이 사용
로그인 정보 저장에 많이 사용
-> 쿠키처럼 토큰(=인증키) 저장하는 방식이 아닌, 아이디와 같은 정보를 인식하여
탭을 닫거나, 특정 시간이 만료되면 세션에 저장되어있는 정보를 자동 삭제

로컬(local)

브라우저를 닫아도 남아있는 데이터, 설정 값 저장에 사용
즐겨찾기와 같은 용도로 많은 사용

비교표

특징 쿠키 세션 로컬
용량
KB < MB
( KB 보다 MB가 더 큰 용량 )		 4KB 5MB
수명 설정한 시간까지 탭 닫을 때까지
설정한 시간까지 		직접 삭제할 때까지
서버 전송 자동 전송 전송 안됨
( JS로 전송 기능을 작성하면 전송 가능 )
